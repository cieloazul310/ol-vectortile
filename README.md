## 開発

```shell
$ git clone git@github.com:cieloazul310/ol-vectortile.git --depth=1
$ cd ol-vectortile
$ yarn install

# develop
$ yarn run start

# build
$ yarn run build

# format with prettier
$ yarn run format

# deploy to gh-pages
$ yarn run deploy
```
